# TJVP App

Is a Graduation project developed with the Android SDK.

## Features
- Diagnose your knee
- Authentication with Firebase
- Display your diagnosis with AI 
- Display exercise with VR


## Android Development
- TensorFlow Lite
- XML
- Android KTX
- Kotlin Coroutines
- Flow State
- ViewModel
- Navigation
- Room
- Retrofit
- Hilt
- Coil
- DataStore
- Firebase
